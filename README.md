# 面试题：一个带验证码倒计时功能的登录组件

## 介绍

直接用原生JavaScript完成的，用到了[Font Awesome](http://fontawesome.io/)的两个图标，之外就没有其他库和框架了

## 功能

* 对手机号进行校验
* 在手机号正确的情况下才允许发送验证码
* 计时过程中不允许再次点击
